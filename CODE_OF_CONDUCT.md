# Code of Conduct

## Our Standard

Contributors should be respectful, direct, and constructive. Technical disagreement is welcome when it is focused on evidence, project goals, and maintainability.

Unacceptable behavior includes harassment, personal attacks, discriminatory language, publishing private information, or pressuring maintainers to accept unlicensed data, model weights, or unsafe agent behavior.

## Scope

This code of conduct applies to project issues, pull requests, discussions, documentation, and related community spaces.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or restrict participation when behavior harms the project or community. Report concerns privately to the repository maintainers.
